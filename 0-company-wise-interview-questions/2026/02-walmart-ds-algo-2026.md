## Below are other frequently asked questions including follow-ups:


### 1. Minimum Cost to Build a Skilled Team

Each candidate has a hiring cost and may know Java, Python, both languages, or neither. For every value of k, find the minimum cost of a team in which at least k selected candidates know Java and at least k know Python.

[https://codezym.com/question/283-min-cost-to-build-skilled-team](https://codezym.com/question/283-min-cost-to-build-skilled-team)


### 2. Reverse a Character List/Array Recursively

Reverse the supplied characters recursively while preserving every character and returning them in the opposite order.

[https://codezym.com/question/284-reverse-character-list-recursively](https://codezym.com/question/284-reverse-character-list-recursively)


### 3. Can Attend All Scheduled Meetings

Given a collection of meeting intervals, determine whether one person can attend every meeting without any time overlap.

[https://codezym.com/question/285-can-attend-all-scheduled-meetings](https://codezym.com/question/285-can-attend-all-scheduled-meetings)


### 4. Delayed Retry Scheduling for Failed Events

Schedule retries for failed events after the required delay and process eligible retries in a consistent order.

[https://codezym.com/question/286-delayed-retry-failed-events](https://codezym.com/question/286-delayed-retry-failed-events)


### 5. Refund Rate Limiter for Merchants

Implement a per-merchant refund rate limiter using a rolling five-minute window. Only allowed refunds count toward the limit; rejected attempts are not recorded.

[https://codezym.com/question/287-refund-rate-limiter-for-merchants](https://codezym.com/question/287-refund-rate-limiter-for-merchants)


### 6. Design Feature Toggle Service

Design a feature-toggle service that can add, enable, disable, and remove features. Feature access may depend on a user's subscription and region.

[https://codezym.com/question/288-feature-toggle-service](https://codezym.com/question/288-feature-toggle-service)


### 7. Move Zeroes and Count Minimum Writes

Move every zero to the end while preserving the order of non-zero values, and calculate the minimum number of writes required to produce the result.

[https://codezym.com/question/289-move-zeros-count-minimum-writes](https://codezym.com/question/289-move-zeros-count-minimum-writes)


### 8. Convert String to Equilateral Triangle Format

Arrange the characters of a string into centered rows that form an equilateral-triangle format while preserving the required character order.

[https://codezym.com/question/290-string-to-equilateral-triangle-format](https://codezym.com/question/290-string-to-equilateral-triangle-format)


### 9. Minimum Moves to Gather Balls in Boxes

Given boxes that may contain balls, calculate for every destination box the total number of adjacent moves required to bring all balls into that box.

[https://codezym.com/question/292-min-moves-gather-balls-in-boxes](https://codezym.com/question/292-min-moves-gather-balls-in-boxes)


### 10. Longest Sequence of Consecutive Integers

Given an unsorted collection of integers, return the length of the longest sequence whose values are consecutive.

[https://codezym.com/question/293-longest-sequence-of-consecutive-integers](https://codezym.com/question/293-longest-sequence-of-consecutive-integers)


### 11. Count Number of Distinct Islands in Grid

Count how many different island shapes exist in a binary grid. Islands with the same shape after translation are considered identical.

[https://codezym.com/question/190-count-distinct-islands-grid](https://codezym.com/question/190-count-distinct-islands-grid)


### 12. Walls and Gates with the Most Assigned Rooms

Find the nearest reachable gate for empty rooms in a grid and determine which gate is assigned the greatest number of rooms.

[https://codezym.com/question/280-walls-and-gates-with-most-assigned-rooms](https://codezym.com/question/280-walls-and-gates-with-most-assigned-rooms)


### 13. Next Greater Element in a Circular List

For each value in a circular list, return the first greater value encountered while moving forward, or the required missing value when no greater element exists.

[https://codezym.com/question/295-next-greater-element-in a circular list](https://codezym.com/question/295-next-greater-element-in%20a%20circular%20list)


### 14. Maximum Stock Profit: At Most K Transactions

Given the stock price for each day, return the maximum profit obtainable with at most k buy-and-sell transactions. A stock must be sold before another one is bought.

[https://codezym.com/question/297-max-stock-profit-k-transactions](https://codezym.com/question/297-max-stock-profit-k-transactions)


### 15. Reverse Characters in Each Word of String

Reverse the characters of every word while preserving the original word order and spaces.

[https://codezym.com/question/298-reverse-character-each-word-of-string](https://codezym.com/question/298-reverse-character-each-word-of-string)

Sort positive, negative, and decimal real-number strings numerically without converting their values to an arithmetic data type. Preserve the original order of equal values.

[https://codezym.com/question/299-sort-decimal-number-strings](https://codezym.com/question/299-sort-decimal-number-strings)
